添加商店菜单直接选择购买物品类：

"碧血残阳\feature\dealer.c"里的do_list() 命令可以得到显示商店物品的代码，从
"碧血残阳\cmds\usr\inventory.c"里可以得到显示身上所带物品的代码,之后就可以在
"碧血残阳\cmds\std\actionavailable.c"里添加合适位置添加TOMUD的菜单代码.这样
就可以完成在商店里的(店小二)身上单击出现菜单上直接出现购买各种物品的菜单项了:)



   其他的一部分特殊人物＆＆物品的菜单,如果没有的,可以效仿我在"碧血残阳\cmds\std\
actionavailable.c"里添加了个别特殊人物的特殊菜单的例子如<朱半仙任务>的菜单,和个
别商店人物身上菜单的列子,我已经做了详细的例子注释和说明,只要在我标注的地方添加
特殊的菜单就可以了,大家可以做为参考:)
